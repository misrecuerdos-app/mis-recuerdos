# Recuerdos — Design Book v1.0

## Propósito

Recuerdos debe transmitir una emoción cálida, elegante y cercana.

La aplicación no debe sentirse como un formulario para subir archivos, sino como una experiencia diseñada para conservar momentos importantes.

---

## Referencia visual oficial

El collage de pantallas aprobado es el Master Design de Recuerdos.

Todas las experiencias deben respetar:

- Composición
- Jerarquía visual
- Espaciados
- Navegación inferior
- Tarjetas con fotografías
- Colores
- Tipografía
- Botones
- Sensación general

Las imágenes pueden cambiar según el evento, pero la estructura visual debe permanecer.

---

## Personalidad

Recuerdos es:

- Elegante
- Cercana
- Emotiva
- Moderna
- Sencilla
- Discreta

Recuerdos no debe sentirse:

- Fría
- Técnica
- Complicada
- Exagerada
- Infantil
- Invasiva
