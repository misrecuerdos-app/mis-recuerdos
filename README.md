# Mis Recuerdos

Aplicación web para recopilar fotografías y videos de eventos mediante un código QR.

Los invitados pueden compartir sus recuerdos desde su celular sin instalar ninguna aplicación, mientras que el organizador conserva todos los archivos en su propia cuenta de Google Drive.

## Estado del proyecto

Versión 1.0 (en desarrollo)

## Objetivo

Crear una plataforma sencilla, rápida y privada para recopilar los recuerdos de:

- XV años
- Bodas
- Graduaciones
- Bautizos
- Cumpleaños
- Eventos sociales
